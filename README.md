ESPAÑOL
------------------------------------------------------------------------------------------------------------------
Autores: Jorge Fuster Cutanda / Bartomeu Alcover Borràs

Archivos:
- dataset: carpeta que contiene los CSV resultantes de utilizar el código indicando que queremos obtener los datos de 2005
- source: carpeta que contiene el código Python main.py
- main.py: código a ejecutar para la obtención de datos
- README.md: documento explicativo con los nombres de los integrantes del grupo, breve descripción de los archivos del repositorio, instrucciones para usar el código y DOI de Zenodo del dataset generado.
- requirements.txt: archivo con las librerías necesarias para ejecutar el código

Funcionamiento:
Extrae los líderes estadísticos de la NBA en temporada regular y Playoffs de la Wikipedia del año que indique el usuario.

Para ejecutar el script es necesario instalar las siguientes bibliotecas:

pip install pandas
pip install requests
pip install lxml
pip install beautifulsoup4

Al ejecutar el script, este pedirá al usuario que introduzca el año del que desea obtener los datos (por ejemplo, si introduce 2005, obtendrá los datos de la temporada 2004-2005, tal y como puede comprobarse en la carpeta dataset de este repositorio). Los datos se almacenan en dos archivos CSV, uno para temporada regular y otro para Playoffs, en una carpeta 'Resultados' (si la carpeta no existe, el script creará una).

DOI de Zenodo con el dataset generado: https://doi.org/10.5281/zenodo.19438712


ENGLISH
-------------------------------------------------------------------------------------------------------------------
Authors: Jorge Fuster Cutanda / Bartomeu Alcover Borràs

Files:
- dataset: folder containing the CSV files generated by running the code to retrieve data from 2005
- source: folder containing the Python code main.py
- main.py: code to be run to retrieve data
- README.md: explanatory document containing the names of the group members, a brief description of the repository files, instructions for using the code, and the Zenodo DOI for the generated dataset.
- requirements.txt: file containing the libraries required to run the code

Functionality:
Extracts the NBA statistical leaders for the regular season and Playoffs from Wikipedia for the year specified by the user.

To run the script, you need to install the following libraries:

pip install pandas
pip install requests
pip install lxml
pip install beautifulsoup4

When the script is run, it will ask the user to enter the year for which they wish to obtain the data (for example, if you enter 2005, you will obtain data for the 2004–2005 season, as can be seen in the ‘dataset’ folder of this repository). The data is stored in two CSV files, one for the regular season and one for the Playoffs, in a “Resultados” folder (if the folder does not exist, the script will create one).

Zenodo DOI for the generated dataset: https://doi.org/10.5281/zenodo.19438712
