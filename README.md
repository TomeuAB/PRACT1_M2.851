ESPAÑOL
------------------------------------------------------------------------------------------------------------------
Extrae los líderes estadísticos de la NBA en temporada regular y Playoffs de la Wikipedia del año que indique el usuario.

Para ejecutar el script es necesario instalar las siguientes bibliotecas:

pip install pandas
pip install requests
pip install lxml
pip install beautifulsoup4

Al ejecutar el script, este pedirá al usuario que introduzca el año del que desea obtener los datos (por ejemplo, si introduce 1989, obtendrá los datos de la temporada 1988-1989). Los datos se almacenan en dos archivos CSV, uno para temporada regular y otro para Playoffs, en una carpeta 'Resultados' (si la carpeta no existe, el script creará una).



ENGLISH
-------------------------------------------------------------------------------------------------------------------
Retrieves the NBA statistical leaders for the regular season and playoffs from Wikipedia for the year specified by the user.

To run the script, you need to install the following libraries:

pip install pandas
pip install requests
pip install lxml
pip install beautifulsoup

When you run the script, it will ask you to enter the year for which you wish to retrieve the data (for example, if you enter 1989, you will receive data for the 1988–1989 season). The data is saved in two CSV files, one for the regular season and one for the play-offs, in a folder called ‘Resultados’ (if the folder does not exist, the script will create one).
